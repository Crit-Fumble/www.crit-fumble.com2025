
#### Custom Creature Creation System
**Extends**: D&D 5e SRD creature rules

This is the CFG 5e-specific creature creation system that allows for more granular character and NPC progression.

**Humanoid Creature Class**:

- **Level 0 (-100 XP) - Trainee Class**:
  - [ ] Implement "Trainee Class" for each Character Class
    - Represents natural ability before formal training
    - Size category determines Hit Die (Tiny d4, Small d6, Medium d8, Large d10, Huge d12, Gargantuan d20)
    - Average-rounded-up max HP based on size
    - "Unarmed Strike" action (Strength to hit, 1 + Strength damage)
    - Can be replaced completely by character class at Level 1

- **Level 1/8 (-90 XP)**:
  - [ ] Background selection at CR 1/8
    - Early background benefits
    - Skill proficiencies

- **Level 1/4 (-75 XP)**:
  - [ ] (Define features for this sub-level)

- **Level 1/2 (-50 XP) - Class Selection**:
  - [ ] Character Class selection
    - Hit Die becomes class Hit Die
    - Max HP becomes starting HP for class
    - Gain class proficiencies
    - If full spellcasting class: gain 1 cantrip and 1 1st-level spell (1/day)

- **Level 1 (0 XP) - Full Character**:
  - [ ] Gain first class level features
  - [ ] Origin Feat selection (SRD feats = "Humanoid" feats)
  - [ ] Ability score determination (same as player characters)
  - [ ] Alignment selection (9 alignments)

**Beast Creature Class**:
- [ ] Age category system
  - Lifecycle: egg/gestation → infant → child → young → adult → ancient → elderly
  - Playable age categories determined by species
  - Age affects size, stats, and abilities
- [ ] Beast types: fish, invertebrate, amphibian, bird, mammal
- [ ] Species selection (dictated by creature class)
  - Controls lifecycle, aging timespan, max growth
  - Playable species have speed ≥ 10 ft
- [ ] Origin Feat specific to Beast Creature Class
- [ ] Ability score determination by creature class
- [ ] Alignment within one category of species alignment

**Plant Creature Class**:
- [ ] Age category system (seed/gestation → infant → child → young → adult → ancient → elderly)
- [ ] Plant types: fungi, moss, fern, flower, conifer
- [ ] Species selection with lifecycle control
  - Playable species have speed ≥ 10 ft
- [ ] Origin Feat specific to Plant Creature Type
- [ ] Ability score determination by creature class
- [ ] Alignment within one category of species alignment

**"Unnatural" Creature Types (WIP - Do Not Iterate Yet)**:
- **Monstrosity Creature Class**: Evolves from beasts via magic or other system effects; can use origin feats from any creature type
- **Ooze Creature Class**: Types: Pudding, Jelly, Slime, etc.; "Awakening" feat: Int, Wis, Cha minimum of 8 for player characters

**"Magical" Creature Types (WIP)**:
- **Giant Creature Class**:
  - [ ] Size progression system
    - Huge giants start as Medium (child) → Large (young) → Huge (adult)
    - Large giants start as Medium (young) → Large (adult)
    - Playable giants: max Medium until Level 11, Large until Level 17, Huge until Level 20
  - [ ] Lineages: Hill, Frost, Fire, Stone, Cloud, Storm, Cyclops, Ogre, Ettin

- **Dragon Creature Class**:
  - [ ] Age category system
    - True Dragons: egg → hatchling → wyrmling → young → adult → ancient → elderly
    - Draconic creatures: Lizardfolk, Kobold, etc.
  - [ ] Species selection (color/type): Red, Black, Gold, Copper, etc.
  - [ ] Suppression system (to play dragons at lower power levels)

**"Constructed" Creature Type**:
- [ ] Construct creature class

**More "Magical" Creature Types**:
- [ ] Aberration
- [ ] Celestial
- [ ] Elemental
- [ ] Fey
- [ ] Fiend
- [ ] Undead
